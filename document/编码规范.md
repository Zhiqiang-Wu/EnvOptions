1. 文件分隔符不允许直接写成 <font color='red'>/</font> 或者 <font color='red'>\\</font> ，应该使用<font color='red'>path.sep</font>
2. 禁止将用于调试的代码提交到git，例如<font color='red'>console.log()</font>，并且打包之前必须要全部删掉
3. 不可使用行后注释
       
       // 正确注释
       const num = 123;
       const num = 123; // 错误注释
4. 无用的代码必须要删掉，不可注释，除非存在特殊用途
